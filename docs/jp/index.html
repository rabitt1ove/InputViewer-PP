プライバシーポリシー

このプライバシーポリシーは、macOSアプリケーション Input Viewer（以下「本アプリ」といいます）に適用されます。本アプリは、Yu（以下「開発者」）によって無料で提供されるユーティリティアプリです。

情報の収集と利用について

本アプリは、個人情報を収集・保存・送信することは一切ありません。

外部サーバーとの通信は行わず、ユーザーの行動を追跡することもありません。

サードパーティサービス

本アプリは、分析、広告、追跡などのサードパーティサービスを一切使用していません。

子どものプライバシー

本アプリはデータを収集しないため、すべての年齢の方に安全にご利用いただけます。13歳未満のお子様から故意に情報を収集することもありません。

セキュリティ

ユーザーの情報を保存または送信しないため、特別なセキュリティ対策は不要です。本アプリはMac上でローカルに動作し、インターネットへの接続も行いません。

ポリシーの変更について

本ポリシーの内容は、必要に応じて将来的に更新されることがあります。変更がある場合は、公式サポートページまたは公開リポジトリ上にてお知らせします。
